Jun 1, 2021 • 3 min read

반복문

  • for문 : 횟수 제어
  • while문 : 조건에 따라 변경!!! 무한루프에 빠질 수 있다.

for

  • 주어진 자료를 순회할때 주로 사용되는 반복구조
for i in [1,2,3,4,5,6,7]:
    print(i)

for i in 'python data':
    print(i)

범위 > range() 함수

  • 시작값/증가값 생략 가능
  • 끝값 생략 불가
for i in range(1,6):
    print(i)

for i in range(5):
    print(i,end= " ") # end='' 줄이 바뀌지 않고 출력된다

for i in range(1,6):
    print(i,end= "  ") # 공백을 지정하면 공백만큼의 간격을 두고 출력한다

for i in range(1,10,2):
    print(i,end= "\t")  # \t 탭 간격으로 출력된다.

for i in range(1,10,-1):
    print(i,end= " ")

for i in range(10,0,-1):
    print(i,end= " ")

for i in range(100):
    print('Hello World!!')

  • 조건식의 결과가 True 동안 반복 / False가 되면 반복문을 빠져나옴
  • while 조건식: 명령블럭
i = 0 
#조건식에 사용되는 변수는 while문에 들어가기 전에 초기화 되어야 한다.

while i <= 30:
    print(i,end= ' ')
    i = i + 3

for i in range(0,31,3):
    print(i,end=' ')

예제 > up & down 게임

  1. 1~100 사이의 정답 숫자를 랜덤으로 하나 정하고
  2. 정답 숫자를 맞출 때까지 숫자를 입력하는 게임.
  3. 입력된 숫자가 정답보다 작으면 'Down', 크면 'Up'을 출력하고
  4. 숫자를 다시 입력받는다.
  5. 정답을 맞추면 정답!이라고 출력하고 게임을 끝낸다
import random
answer = random.randint(1,100)
num = 0
while num != answer:
    num = int(input('숫자를 입력하세요: '))
    if num > answer:
        print('Down')
    elif num < answer:
        print('Up')
print(f'정답! 답은 {answer}입니다!')
